# Security Policy

We take the security of our users seriously. If you have found a security vulnerability in Scratch Copilot AI, please report it privately.

## Reporting a Vulnerability

Please do not open a public issue. Instead, send an email to **security@scratch-copilot.org** (or equivalent contact).

Include the following in your report:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

We will aim to acknowledge your report within 48 hours and work with you to resolve the issue as quickly as possible.
